# get_frequencies

The `get_frequencies` function should take a string and return a dictionary that represents frequencies of each character in the string. It should ignore spaces.

## Example

```py
get_frequencies('hello world')

# {
#   h: 1,
#   e: 1,
#   l: 3,
#   o: 2,
#   w: 1,
#   r: 1,
#   d: 1
# }

```
